# Music Website Application

A browser-based music player built using HTML, CSS, and JavaScript. The application allows users to play, pause, and navigate through songs while demonstrating the practical use of a Doubly Linked List data structure for playlist management.

## Features

* Play and pause music tracks
* Navigate to the next song
* Return to the previous song
* Interactive playlist management
* Responsive and user-friendly interface
* Doubly Linked List implementation for efficient song navigation

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```text
Music_Website_Application/
├── README.md

└── utils.js
```

## How It Works

The application uses a Doubly Linked List to manage the playlist. Each song is represented as a node containing references to both the previous and next songs, enabling smooth forward and backward navigation.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Swaroop-Haridas/Music_Website_Application.git
```

2. Navigate to the project directory:

```bash
cd Music_Website_Application
```

3. Open `index.html` in your preferred web browser.

No additional dependencies or installation steps are required.

## Usage

* Click the play button to start playback.
* Use the next button to move to the next track.
* Use the previous button to return to the previous track.
* Browse the playlist and select songs as needed.
